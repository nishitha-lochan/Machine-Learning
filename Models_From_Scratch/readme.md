# ML Models From Scratch (Python)

This repository is a **long-term collection of Machine Learning algorithms implemented completely from scratch** using Python. The goal is to deeply understand how machine learning works internally by building algorithms step by step, without relying on high-level ML libraries.

The focus of this repository is on **fundamentals, clarity, and learning**, rather than quick results using pre-built functions.
